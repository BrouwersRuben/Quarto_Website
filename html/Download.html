Downloading our game!

Thank you so much for making it this far, and being interested in downloading our game!

The first step in downloading our game is clicking the first button labeled "Quarto JAR file" and downloading the file that is attached to it.

Quarto JAR file

After downloading this file, you will also need JavaFX on your system to run our game. This you can donwload from this link.
(You will need a JavaFX from the version 11 or higher, and also a correct version of the Java Runtime Environment) You just select your operating system, and run the executable. Make sure you remember in which directory you save the program files, because you will need the path to the lib folder later.

Next step on the list is to download the zip file with the wallet for our database, this is just a zip file which you have to extract into a map with the JAR file, so you should have a directory with the JAR file and the extracted zip folder of the wallet.

For the wallet file: Please contact us.

Then you should open the command line from withing the directory with the JAR file and the Database wallet, and type this into the command line:

java \
                --module-path YOUR PATH/javafx/javafx-sdk-11.0.2/lib 

                (Your version may differ) 

                --add-modules=javafx.controls,javafx.media 

                -jar quarto.jar

After this you should be able to succesfully run the game. If this is not the case, please contact us and we will walk you through the installation.
